# AGENTS.md

This repository contains a command-line coding agent for reading and modifying git repositories with local tools.

When working in this project:
- Prefer reading `README.md` first when asked what the project does.
- Use the built-in tools when they help: `ls`, `cat`, `grep`, `calculate`, `compact`, `doctests`, `write_file`, `write_files`, `rm`, and `pip_install`.
- Never use absolute paths or paths containing `..`.
- Keep edits as small and reviewable as possible.
- Use clear commit messages because agent changes are recorded in git.
